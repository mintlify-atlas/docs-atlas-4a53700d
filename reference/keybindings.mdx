This page documents all keybindings configured in nvim-luffy. The leader key is set to `Space`.

## General keybindings

### Window management

| Keybinding | Mode | Description |
|------------|------|-------------|
| `<leader>\|` | Normal | Split window vertically |
| `<leader>sh` | Normal | Split window horizontally |

### Buffer management

| Keybinding | Mode | Description |
|------------|------|-------------|
| `<leader>bd` | Normal | Close current buffer |
| `<leader>ba` | Normal | Close all buffers |

<Note>
The keybindings for exiting insert mode (`jk`) and clearing search highlights (`<leader>nh`) are commented out in the default configuration.
</Note>

## LSP keybindings

These keybindings are available when an LSP server is attached to the current buffer.

### Navigation

| Keybinding | Mode | Description | Source |
|------------|------|-------------|--------|
| `gR` | Normal | Show LSP references | lua/leyland/lsp.lua:11 |
| `gd` | Normal | Go to declaration/definition | lua/leyland/lsp.lua:14 |
| `gi` | Normal | Show LSP implementations | lua/leyland/lsp.lua:20 |
| `gt` | Normal | Show LSP type definitions | lua/leyland/lsp.lua:23 |

### Code actions

| Keybinding | Mode | Description | Source |
|------------|------|-------------|--------|
| `<leader>ca` | Normal, Visual | See available code actions | lua/leyland/lsp.lua:26 |
| `<leader>rn` | Normal | Smart rename | lua/leyland/lsp.lua:29 |
| `K` | Normal | Show documentation for symbol under cursor | lua/leyland/lsp.lua:48 |

<Info>
In visual mode, `<leader>ca` will apply code actions to the selected text.
</Info>
