## Welcome to nvim-luffy

nvim-luffy is a carefully crafted Neovim configuration that transforms your editor into a powerful, modern development environment. Built on top of the lazy.nvim plugin manager, it provides everything you need for productive coding right out of the box.

## Why nvim-luffy?

nvim-luffy combines the speed and efficiency of Neovim with a curated selection of plugins and sensible defaults. Whether you're working with TypeScript, Python, Lua, or any modern language, you'll have powerful tools at your fingertips from day one.

## Core features

### Smart plugin management

Built on [lazy.nvim](https://github.com/folke/lazy.nvim), nvim-luffy loads plugins only when needed, ensuring fast startup times even with a rich plugin ecosystem.

### Complete LSP integration

Powerful language server support through Mason and nvim-lspconfig provides:
- Automatic installation of language servers
- Code completion with context-aware suggestions
- Go-to-definition, find references, and rename refactoring
- Real-time diagnostics and error checking
- Support for TypeScript, Python, Lua, HTML, CSS, and more

### Fuzzy finding with Telescope

Quickly navigate your codebase with blazing-fast fuzzy finding:
- Find files by name (`<leader><leader>`)
- Search across all files (`<leader>/`)
- Browse recent files (`<leader>fr`)
- Find TODOs and comments (`<leader>ft`)

### Intelligent code navigation

**Harpoon** lets you mark and instantly jump between your most important files using `<leader>H` to mark and `<leader>1-9` to jump.

**nvim-tree** provides a file explorer with git integration, accessible via `<leader>e`.

### Git superpowers

- **Neogit**: A magit-inspired git client (`<leader>gg`)
- **Diffview**: Beautiful diff views for comparing changes (`<leader>dd`)
- Visual branch comparisons and merge conflict resolution

### Modern code editing

<Info>
nvim-luffy includes smart defaults for a pleasant editing experience.
</Info>

- **Auto-completion**: Context-aware suggestions with nvim-cmp
- **Syntax highlighting**: Treesitter-powered highlighting for 20+ languages
- **Auto-formatting**: Format-on-save with Conform (Prettier, Stylua, Black)
- **Auto-pairs**: Automatic bracket and quote pairing
- **Flash**: Lightning-fast cursor movement with search labels

### Tmux integration

Seamless navigation between Neovim splits and Tmux panes with `vim-tmux-navigator`, making your terminal workspace feel unified.

### Beautiful aesthetics

Choose from multiple premium colorschemes:
- Bamboo (default)
- Catppuccin
- Tokyo Night
- Rose Pine
- Modus Themes
- OneDark

## Philosophy

nvim-luffy embraces these principles:

1. **Sensible defaults**: Works great out of the box without requiring hours of configuration
2. **Modern tooling**: Uses the latest and most maintained plugins in the ecosystem
3. **Performance first**: Fast startup and responsive editing through lazy loading
4. **Developer focused**: Optimized for professional software development workflows
5. **Extensible**: Easy to customize and extend with your own preferences
